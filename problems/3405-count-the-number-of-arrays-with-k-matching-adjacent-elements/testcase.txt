示例 1：

输入：n = 3, m = 2, k = 1

输出：4

解释：

总共有 4 个好数组，分别是 [1, 1, 2] ，[1, 2, 2] ，[2, 1, 1] 和 [2, 2, 1] 。
所以答案为 4 。

示例 2：

输入：n = 4, m = 2, k = 2

输出：6

解释：

好数组包括 [1, 1, 1, 2] ，[1, 1, 2, 2] ，[1, 2, 2, 2] ，[2, 1, 1, 1] ，[2, 2, 1, 1] 和 [2, 2, 2, 1] 。
所以答案为 6 。

示例 3：

输入：n = 5, m = 2, k = 0

输出：2

解释：

好数组包括 [1, 2, 1, 2, 1] 和 [2, 1, 2, 1, 2] 。
所以答案为 2 。
